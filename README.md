# discordIFAbot
A discord bot utilizing discordpy, Google Sheets API, as well as many others to help with automating the IFA discord in bookkeeping and other experiences.

# The main goal
## - The main goal of this bot is to do the following:
### o Ingest all members in the discord server into a database.
### o Organize information in a readable format, such as google sheets.
### o Alert members to promotions.
### o Remove members from Google Sheets.
