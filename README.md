# discordIFAbot
A discord bot utilizing sqlite3, discordpy, and google spreadsheets API.

# The main goal
## - The main goal of this bot is to do the following:
### o Ingest all members in the discord server into a database.
### o Organize information in a readable format, such as google sheets.
### o Alert members to promotions.
### o Remove members from Google Sheets.

# What is this program?
This program is for a discord bot that records the users of the Inquisitorial Force Apolloyon discord server, an Arma 3 community focused on Warhammer 40k. It is a simple program and can be repurposed by others for their own needs. If you decide to repurpose this program, please make sure to replace the token key with your own .env file.
